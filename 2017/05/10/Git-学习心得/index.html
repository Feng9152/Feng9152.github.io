Git 学习心得

发表于 2017-05-10 | | 阅读次数

Git 是开发中使用最广泛的分布式版本控制系统（没有之一）。
不同于 SVN 等集中式版本控制系统，Git 使用时候不需要时刻联网，因为每个人电脑里都有完整的版本库，所以其安全性也要高很多。Git 还有强大便捷的远程仓库、分支管理等功能！赢得了全球开发者的青睐！

创建版本库

  • git init 初始化 git 仓库
  • git add 添加文件到 git 仓库
  • git commit -m “xxxx”
  • git status 查看工作去区状态
  • git diff 查看文件修改内容

    项目版本管理

  • git log 查看提交历史确定回退版本
  • git log –pretty=oneline
  • git reflog 查看命令历史确定前进到哪个版本
  • git reset –hard commit_id
  • 暂时储藏工作现场
  • git stash
    重回工作现场
  • git stash pop
    丢弃没有合并过的分支
  • git branch -D
    丢弃工作区修改
  • git checkout – file
  • git reset HEAD file
    删除一个文件
  • git rm

    远程仓库

    关联远程库
  • git remote add origin git@server-name:path/repo-name.git
    推送分支内容
  • git push -u origin master
  • git push origin master
    克隆远程库
  • git clone

    分支管理

  • git branch 查看当前分支
  • git branch 创建分支
  • git checkout 切换分支
  • git checkout -b 创建 + 切换分支
  • git merge 合并某分支到当前分支
  • git branch -d 删除分支
  • git log –graph 查看分支合并图

    标签管理

  • git tag 新建一个标签
  • git tag -a -m “xxxxx” 指定标签信息
  • git tag 查看所有标签
  • git push origin
  • git push origin –tags
  • git push origin :refs/tags/

    参考信息
